# Портфолио разработчика

Современное и адаптивное портфолио, построенное на **Vue 3 + Vite**. Проекты загружаются из Markdown-файлов,
поддерживаются фильтрация по тегам, тёмная тема, PWA и автоматический деплой на GitHub Pages.

🔗 **Демо:** [https://axelmark.github.io](https://axelmark.github.io)  
📁 **Репозиторий:** [https://github.com/axelmark/axelmark.github.io](https://github.com/axelmark/axelmark.github.io)

---






## 🛠️ Стек технологий

- **Vue 3** – Composition API, `<script setup>`
- **Vite** – быстрая сборка и разработка
- **Vue Router** – маршрутизация
- **Markdown** – контент проектов с frontmatter (без внешних библиотек)
- **Highlight.js** – подсветка кода
- **CSS** – переменные для темной/светлой темы, адаптивная сетка
- **GitHub Actions** – автоматический деплой на GitHub Pages
- **PWA** – возможность установки на устройство
